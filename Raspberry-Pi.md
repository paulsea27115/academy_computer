## Raspberry-Pi 한글다운
   ```
   sudo apt-get -y update && sudo apt-get -y  upgrade
   
   sudo apt install -y fonts-unfonts-core
   
   sudo apt install ibus-hangul
   
   sudo reboot
   ```

   ```
   sudo reboot
   ```
